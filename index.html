项目

Math Verifier Agent

基于 LangChain 和 LangGraph 构建的智能体系统，能够自动验证数学竞赛题目的解答答案是否正确。通过整合多个工具和推理模块，实现了高效的数学问题分析和验证流程。该项目展示了如何将 AI 模型与结构化推理相结合来解决复杂的数学问题。

技术栈：LangChain / LangGraph / Python / LLM

Transformer 从零实现

从头完整实现 Transformer 模型架构，包括多头自注意力、前向网络、位置编码等核心组件。通过这个项目深入理解了 Attention 机制的原理，并在实际实现中掌握了参数初始化、数值稳定性等工程细节。代码注释详细，可用于学习和教学。

技术栈：PyTorch / Python / 深度学习
